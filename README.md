# claude-orchestration

## Файлы

### .claude.json (копия)

Копия файла `~/.claude.json`. Оригинал находится в корне домашней директории — это жёсткое требование Claude Code, перенести его в `~/.claude/` нельзя. Копия хранится здесь для справки. В будущем при настройке конкретного проекта этот файл нужно вынести в корень проекта.

### mcp.json

Справочная копия конфигурации MCP-серверов. Хранится здесь для наглядности и version control.

**Реальное место чтения:** `~/.claude.json` (ключ `mcpServers` на верхнем уровне) — единственный официально поддерживаемый путь для глобальных MCP-серверов в Claude Code. Работает везде: VSCode extension, CLI из любой директории.

`~/.claude/mcp.json` Claude Code **не читает** — этот файл служит только документацией.

При настройке новой машины нужно вручную скопировать содержимое `mcpServers` из этого файла в `~/.claude.json`.
