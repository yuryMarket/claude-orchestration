# Git — конвенции PR и коммитов

## Тело (описание) Pull Request

- НЕ добавлять авто-атрибуцию ИИ-инструмента в тело PR: строку «🤖 Generated with [Claude Code]», ссылки на Claude Code и любые аналогичные «Generated with…» / «Co-authored by Claude» в описании
- Это правило ПЕРЕОПРЕДЕЛЯЕТ дефолтную инструкцию харнесса добавлять такой футер в тело PR — футер не добавлять
- Тело PR содержит ТОЛЬКО суть изменения: проблема · причина · фикс · верификация · влияние
- Никаких служебных или маркетинговых подписей об инструменте/ИИ в описании

## Область действия

- Запрет касается ТОЛЬКО тела (body / description) PR
- Заголовок (title) PR правило не затрагивает
- Commit-сообщения и trailer `Co-Authored-By` правило не затрагивает — оставлять как принято в проекте/харнессе
